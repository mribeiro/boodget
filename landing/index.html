Everything you need

All your finances,
one dashboard

From monthly snapshots to expense cycles and long-term goals — boodget covers the full picture without making you feel like an accountant.

📈
Capital Snapshots
Record account values every month and watch your net worth grow over time. Track investments, savings accounts, and current accounts with a beautiful evolution chart.
💸
Monthly Expense Cycles
Model your real budget month by month. Fixed expenses with payment dates, budget items with spending limits, and salary + previous balance tracking.
🎯
Financial Goals
Set targets with deadlines. Automatically track progress via distributions, manual contributions, or historical data. Visual progress bars and timeline charts included.
🔭
Glances Dashboard
An at-a-glance panel shows Capital, Current Cycle balance, Next Expense, Goals status, and Emergency Fund health — colour-coded with amber and red urgency alerts.
🛡️
Emergency Fund Tracker
Define a savings multiplier, pick contributing accounts, and add extra monthly expenses. Instantly know if you're covered for the recommended N months of expenses.
📅
Annual Expenses
Track annual bills with multiple installments across months. A carryover balance, payment tracking, and cycle integration keep big-ticket expenses under control.
🧮
Workbench Scenarios
Model income vs. expenses vs. distributions with Must/Want/Save breakdowns before committing. Save named snapshots and compare scenarios side by side.
🔔
Push Notifications
Get notified N days before upcoming fixed expenses. Configurable delivery times, repeat intervals, and per-dossier opt-in. Works via browser push (no app needed).
👥
Multi-User Sharing
Share any dossier with other users. Multiple dossiers per account, fine-grained access control, and optional OIDC/SSO support for teams.
Monthly Expenses

Budget cycles that match how you actually spend

Most people's months don't align with the calendar. Capital Tracker lets you define a custom start day (e.g. 25th), so each cycle covers your real pay period — not an arbitrary Jan 1 boundary.

  • Fixed expenses with per-item payment dates and "paid" checkboxes
  • Budget items with max spending and tracked "spent" amounts
  • Salary + previous balance → real leftover calculation
  • Templates auto-copy into each new cycle with one click
  • Mark cycles as closed with a confirmed final balance

Financial Goals

Track what matters with a clear finish line

Whether you're saving for a home, an emergency fund, or a dream trip — Goals give you a target amount, a deadline, and automatic progress tracking tied to your real financial activity.

  • Progress tracked via distribution items, manual entries, or ad-hoc amounts
  • Historical contributions for months before cycle tracking started
  • Auto-computed state: Active, Completed, or Failed
  • Timeline chart showing actual vs. projected trajectory
  • Extra value with impact mode (reduce monthly or anticipate end date)

Self-Hosted · Open Source

Your data stays where it belongs — with you

boodget runs on your own server. One Docker command and you're up. No subscriptions, no cloud account, no one looking at your spending habits but you.

Node.js backend
React 18 frontend
SQLite database
Docker deployment
PWA installable
OIDC / SSO ready
Zero dependencies cloud
# Clone & deploy in minutes
$ git clone https://github.com/ViBE-MiNDS/capital-tracker
$ cd capital-tracker
# Edit SESSION_SECRET in docker-compose.yml
$ docker compose up --build -d
✓ capital-tracker started
✓ Listening on http://localhost:3000
✓ SQLite database ready at /data/
# That's it. Not scary at all.
How it works

Up and running in four steps

1
Deploy
Run one Docker command on any server or home lab. SQLite stores everything locally — no external database needed.
2
Configure
Create a dossier, add your accounts, set a cycle start day, and define your expense template once.
3
Track Monthly
Open a new cycle each month, record account snapshots, and mark expenses paid as the days go by.
4
Stay Informed
Let Glances, push notifications, and Goals keep you on top of your financial situation — automatically.
